# A Sidescrolling boss rush game!!

## What is this?
this is a **simple**, almost even a MVP (minimum viable product) of a boss rush
game. i made this in my free time and now just barely functions.
this is by no means *finished*, but it is **playable**

theres not even healing yet lol

have fun!

## how do i play it?
### controls
| keys | description |
| -------------- | --------------- |
| A & D | move left and right |
| Spacebar | dodge roll (you are invulerable untile the animation ends!) |
| M | slash. your attack. |
